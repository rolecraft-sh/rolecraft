## Before Opening an Issue

1. Check the [documentation](docs/) to see if your question is already answered
2. Search [existing issues](https://github.com/rolecraft-sh/rolecraft/issues) for similar problems
3. Run `rolecraft doctor` to diagnose common problems
4. Include your Node.js version (`node --version`) and rolecraft version (`rolecraft --version`)
